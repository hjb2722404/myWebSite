# 代码显示测试

这篇文章用来测试不同编程语言的代码显示效果。

## JavaScript 示例

这是一个简单的 JavaScript 函数：

```javascript
function calculateSum(numbers) {
  return numbers.reduce((sum, num) => {
    // 添加到总和
    return sum + num;
  }, 0);
}

// 测试函数
const numbers = [1, 2, 3, 4, 5];
console.log(calculateSum(numbers)); // 输出: 15
```

## Python 示例

这是一个 Python 类的示例：

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, my name is {self.name} and I'm {self.age} years old!"

# 创建实例
person = Person("Alice", 25)
print(person.greet())
```

## TypeScript 示例

这是一个 TypeScript 接口和类的示例：

```typescript
interface Vehicle {
  brand: string;
  year: number;
  start(): void;
}

class Car implements Vehicle {
  constructor(
    public brand: string,
    public year: number,
    private engine: string
  ) {}

  start(): void {
    console.log(`Starting ${this.brand} ${this.engine} engine...`);
  }
}

const myCar = new Car("Tesla", 2024, "electric");
myCar.start();
```
